jstree 一款jquery 树形目录插件，支持HTML和json格式数据，ajax或异步调用。

官方文档及项目源码请参考：

   https://www.jstree.com/    https://github.com/vakata/jstree
Using HTML
<div id="container">
  <ul>
    <li>Root node
      <ul>
        <li>Child node 1</li>
        <li>Child node 2</li>
      </ul>
    </li>
  </ul>
</div>
<script>
  $(function() {
      $('#container').jstree();
  });
</script>
  • Root node
    • Child node 1
    • Child node 2
Using JSON
<div id="container"></div>
<script>
$(function() {
  $('#container').jstree({
    'core' : {
      'data' : [
        { "text" : "System", "children" : [
            { "text" : "apache" },
            { "text" : "nginx" }
          ]
        },
        { "text" : "Tools", "children" : [
            { "text" : "Idea" },
            { "text" : "eclipse" }
          ]
        }
      ]
    }
  });
});
</script>
